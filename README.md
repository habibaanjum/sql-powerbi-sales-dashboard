End-to-end Business Intelligence project using SQL Server for data transformation and Power BI for interactive dashboard visualization.

#Project Overview
This project analyzes sales transaction data to uncover insights related to revenue trends, customer behavior, and product performance.
Using SQL Server for data preparation and Power BI for visualization, the goal is to transform raw transactional data into meaningful business insights that support decision-making.

#Business Problem
The business needs visibility into:

Overall revenue performance
Monthly sales trends
High-performing products
Regional sales distribution

This dashboard provides a centralized analytical view to monitor key performance indicators (KPIs).

#Tools & Technologies

Microsoft SQL Server (Data querying & preparation)
T-SQL (Data aggregation & KPI calculations)
Power BI Desktop (Dashboard & visualization)
DAX (Measures & time intelligence)

#Data Preparation
Raw sales data was imported into SQL Server and structured into analytical tables.

Using SQL:
Cleaned inconsistent records
Calculated revenue metrics
Aggregated data for reporting
Validated business KPIs before visualization
SQL was used primarily to prepare a clean and analysis-ready dataset.

#Key Insights & KPIs
The dashboard answers the following:

What is the total revenue and monthly growth trend?
Which region contributes the highest sales?
Which products generate the most revenue?
Which customer segment drives the business?

#Dashboard Features
Interactive slicers (Date, Region, Segment)
Revenue trend line chart
Top 5 products visualization
Regional performance comparison
Customer segment breakdown
KPI cards (Total Revenue, Total Orders, Average Order Value)

#Analytical Approach
Data Cleaning & Validation
Revenue Calculation
Aggregation by Time, Region, and Product
Trend & Performance Analysis
Interactive Visualization

#Outcome
This project demonstrates the ability to:
Translate raw data into actionable insights
Perform business-focused SQL analysis
Design interactive dashboards in Power BI
Present KPIs clearly for stakeholders
Support data-driven decision-making

#Sample Use Case
A business manager can use this dashboard to:

Improve revenue forecasting
